# HABRI Revision Tracker

Target journal: **Telecommunications Policy**

Track peer review comments — both synthetic pre-submission reviews and actual journal reviews —
through triage, revision, and archival.

---

## How to use this file

1. **Synthetic pre-submission reviews** — run before first submission using `/review-new` (see `docs/skills.md`)
2. **Actual journal reviews** — paste reviewer comments here after receiving a decision letter
3. **Triage each comment** — Accept / Revise / Decline (with rationale for Decline)
4. **Check off items** as they are addressed in the draft
5. **Archive** the completed cycle to `docs/reviews/archive/` before the next submission

## Pre-Submission Checklist

Run these before initiating the first synthetic review cycle:

- [x] Q1–Q4 2025 quarterly downloads complete — 6 quarters (Q3 2024 – Q4 2025) processed
- [x] Time series analysis script written and run — `scripts/plot_time_series.py` DONE
- [x] Power grid integration decision made — integrated (4-component I_F, Apr 2026)
- [x] All validation figures reflect final data — all maps regenerated Apr 2026 via `scripts/plot_habri_maps.py`
- [ ] Manuscript results/discussion rewritten to reflect final index (F1–F8 flags in article_outline.md)
- [ ] Word count within journal limits (≤ 10,000 body text)
- [ ] Abstract ≤ 250 words
- [ ] All figures ≥ 300 DPI
- [ ] References formatted as APA — F8 citations must be verified before submission
- [ ] Highlights written (3–5 bullets, ≤ 85 characters each)
- [ ] CRediT author contribution statement drafted
- [ ] Data availability statement drafted (GitHub + Zenodo DOI)
- [ ] Supplementary methodology appendix finalized

## Notes on Synthetic Review Focus Order

For *Telecommunications Policy*, recommended sequence before first submission:

1. `policy` — Is the policy relevance and practitioner actionability convincing?
2. `methods` — Is the index construction and validation statistically rigorous?
3. `clarity` — Is the paper accessible to a multidisciplinary readership?
4. `social_sciences` — Is the equity framing (C_C sub-index) theoretically grounded?
